# tree-visualisation

Browser-based visualisation tools converted from the tutorial Java JARs.

## Charts

- DOI tree with DOI XML samples and custom XML upload.
- Treemap with TM3 samples and custom TM3 upload.
- D&C treemap with XML samples, shape presets, and clickable drill-down sections.

## Run Locally

Double-click:

```text
run-doitree-web.bat
```

Or run:

```powershell
node server.js
```

Then open:

```text
http://127.0.0.1:5188/
```

## GitHub Pages

This is a static web app. GitHub Pages can serve it directly from the repository root.

Required files:

- `index.html`
- `styles.css`
- `app.js`
- `data/`
